# Материалы для публикации в Chrome Web Store

Сборка для стора: `npm run zip:store` → архив в `.output/` (вариант **без**
CourseHunter — в манифесте только Udemy). Полная сборка для личного
использования: `npm run build` → `.output/chrome-mv3`.

---

## Название

**Alex Apps Subtitle Translator**

(имя издателя в Developer Console тоже можно поставить «Alex Apps»)

## Краткое описание (до 132 символов)

> Переводит английские субтитры курсов Udemy на украинский и русский: двойные субтитры поверх видео

## Подробное описание

```
Смотрите англоязычные курсы на Udemy с субтитрами на родном языке.

Расширение на лету переводит английские субтитры лекции и показывает их поверх
видео — в режиме двойных субтитров (оригинал + перевод, идеально для изучения
английского параллельно с курсом) или только перевода.

ВОЗМОЖНОСТИ
• Перевод на украинский или русский — язык выбирается в настройках
• Двойные субтитры: английский оригинал + перевод одновременно
• Умный перевод: реплики склеиваются в целые предложения перед переводом,
  поэтому перевод связный, а не пословный
• Кэширование: каждая лекция переводится один раз, повторный просмотр мгновенный
• Два движка: Google Translate (по умолчанию) или локальный Chrome AI
  (перевод не покидает ваш компьютер, Chrome 138+)
• Настройка размера шрифта и режима отображения
• Никакой телеметрии: расширение не собирает данные и не имеет своих серверов

КАК ПОЛЬЗОВАТЬСЯ
1. Откройте лекцию курса на Udemy
2. Расширение само найдёт английские субтитры и переведёт их
3. Язык и режим отображения — в попапе расширения

Расширение не связано с Udemy, Inc. Для работы перевода текст субтитров
отправляется в выбранный сервис перевода (см. политику конфиденциальности).
```

## Ответы для формы «Privacy practices»

**Single purpose description:**
> Translates English course subtitles on Udemy into Ukrainian or Russian and
> displays them over the video player.

**Обоснование разрешений (Permissions justification):**

| Разрешение | Обоснование |
|---|---|
| `storage` | Хранение пользовательских настроек (язык, режим) и локального кэша переведённых субтитров |
| `https://*.udemy.com/*` | Чтение субтитров лекции на страницах Udemy и отображение перевода поверх плеера |
| `https://*.udemycdn.com/*`, `https://udemy-captions.s3.amazonaws.com/*` | Загрузка файлов субтитров (.vtt), на которые ссылается Udemy |
| `https://translate.googleapis.com/*` | Отправка текста субтитров на перевод |

**Data usage disclosures:**
- Website content (subtitle text) → передаётся стороннему сервису перевода;
  не продаётся, не используется для целей, не связанных с основной функцией,
  не используется для оценки кредитоспособности.
- Ничего больше не собирается: не собираются PII, health, financial, auth,
  communications, location, web history, user activity.

**Privacy policy URL:** опубликовать `PRIVACY.md` (например, GitHub Pages или
gist) и вставить ссылку.

## Замечания к ревью (Notes for reviewer)

> The extension reads the lecture's caption track (VTT) that Udemy already
> serves to the logged-in user, translates it via the selected translation
> service, and renders the translation in an overlay. It does not download
> videos, does not bypass DRM, and does not collect any user data.

---

## Известные риски ревью

- Использование неофициального endpoint Google Translate — если ревью придерётся,
  запасной план: перейти на официальный Cloud Translation API с BYOK или свой
  прокси (это же понадобится для премиум-тарифа).
- Приватный api-2.0 Udemy: работает от сессии пользователя, содержимое не
  модифицируется — аналогичные расширения (Udemy Dual Subtitles, 30k+ юзеров)
  давно опубликованы и живы.
