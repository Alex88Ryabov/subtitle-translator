# Alex Apps Subtitle Translator

Chrome-расширение, которое переводит английские субтитры видеокурсов на
**украинский** и **русский** и показывает их поверх плеера — в режиме двойных
субтитров (оригинал + перевод) или только перевода.

Поддерживаемые площадки: **Udemy**, **Coursera** (и CourseHunter в полной сборке).

## Возможности

- 🎬 Двойные субтитры поверх видео или только перевод — переключается на лету
- 🧠 Умный перевод: фрагментированные реплики склеиваются в предложения перед
  переводом, поэтому перевод связный, а не пословный
- ⚡ Кэширование: лекция переводится один раз, повторный просмотр — мгновенно
- 🌐 Два движка: Google Translate (по умолчанию) и локальный Chrome AI
  (Translator API, Chrome 138+ — текст не покидает устройство)
- 🔒 Без телеметрии и своих серверов ([политика конфиденциальности](PRIVACY.md))

## Установка для разработки

```bash
npm install
npm run dev          # Chrome с hot-reload
```

## Сборка

```bash
npm run build        # полная сборка → .output/chrome-mv3
npm run build:store  # сборка для Chrome Web Store (Udemy + Coursera) → .output/store-mv3
npm run zip:store    # zip для загрузки в стор
```

Загрузка распакованного: `chrome://extensions` → «Режим разработчика» →
«Загрузить распакованное» → каталог `.output/chrome-mv3`.

## Проверки

```bash
npm run compile      # tsc --noEmit
npm test             # vitest (парсер VTT, склейка/разбивка предложений)
```

## Как это работает

1. Контент-скрипт получает файл субтитров лекции (WebVTT)
2. Реплики склеиваются в предложения (`lib/sentences.ts`) и батчем переводятся
   выбранным движком (`lib/providers/`)
3. Перевод распределяется обратно по репликам пропорционально длине оригинала
4. Результат кэшируется в `chrome.storage.local` и рендерится в собственном
   shadow-DOM оверлее, синхронизированном с `video.currentTime`

Стек: [WXT](https://wxt.dev) + TypeScript (strict) + React (popup), Manifest V3.
